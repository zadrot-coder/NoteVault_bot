# NoteVault_bot

**NoteVault_bot** — телеграм-бот на **aiogram**, который показывает меню и обрабатывает простые команды/кнопки. Сейчас он умеет:
- показывать главное меню
- отвечать на `/help`
- показывать FAQ по кнопке `📋FAQ`
- реагировать на кнопки `📝Заметки`, `💲Долги`, `🎂Дни рождения`
- возвращать `file_id` для фото и стикеров

## Требования
- Python 3.10+

## Настройка
Перед запуском задай переменную окружения с токеном бота:
```bash
export BOT_TOKEN="ТУТ_ВВЕДИ_СВОЙ_ТОКЕН"
```

## Запуск
Запусти бота одним из вариантов:
```bash
python NoteVault_bot/main.py
```
или
```bash
python -m NoteVault_bot.main
```

После запуска бот перейдёт в режим long polling.

## Команды и кнопки
### Команды
- `/start` — открыть приветственное сообщение и главное меню
- `/help` — ответ-заглушка "Я умею повторять твои сообщения..."

### Текстовые ответы (кнопки меню)
- `📝Заметки` — ответ: раздел заметок
- `💲Долги` — ответ: раздел долгов
- `🎂Дни рождения` — ответ: раздел дней рождения
- `📋FAQ` — вывод FAQ (HTML-разметка)

### Дополнительно
- Сообщение **"Кто автор бота?"** — ответ: `zadrot (Alan)`
- Отправь **фото** — бот вернёт `ID фото` (file_id)
- Отправь **стикер** — бот вернёт `ID стикера` (file_id)

## Примечания
- Токен бота хранится в переменной окружения `BOT_TOKEN` (в `NoteVault_bot/config.py`).
- Бот использует `aiogram` роутер и polling (`NoteVault_bot/main.py`).
