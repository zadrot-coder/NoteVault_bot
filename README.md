# 📦 NoteVault — Personal Digital Vault Telegram Bot

> **[ Python 3.9+ ]** &nbsp;&middot;&nbsp; **[ aiogram 3.x ]** &nbsp;&middot;&nbsp; **[ aiosqlite / SQLite ]**

---

> 💻 **"Только структурированная польза и чистый функционал. Никакой воды".** — *zadrot (Alan)*


**NoteVault** — это современный, полностью асинхронный Telegram-бот, спроектированный как надежное персональное цифровое хранилище. Он позволяет разгрузить голову, мгновенно сохраняя ваши личные заметки, финансовые обязательства (долги) и памятные даты в изолированную базу данных.

---

## 🔥 Ключевые возможности (Features)

*   **📝 Умные заметки:** Быстрое сохранение мыслей, ссылок или списков задач с автоматическим присвоением уникального `ID` для удобного чтения и удаления.
*   **💲 Контроль долгов:** Раздельный учет взаимных расчетов (кто должен вам и кому должны вы), фиксация имени, суммы и текстовых комментариев.
*   **🎂 Менеджер праздников:** Хранение дней рождения ваших близких и друзей, структурированное в виде наглядного списка.
*   **🔒 Локальная безопасность:** Жесткая привязка всех записей к вашему `Telegram ID`. Никто другой не имеет доступа к вашему хранилищу.
*   **🛡️ Защита от спама:** Встроенный интеллектуальный Anti-Flood Middleware, предотвращающий зависание бота при высоких нагрузках.
*   **🛠️ Developer-режим:** Встроенные обработчики, позволяющие на лету вытаскивать `file_id` отправленных стикеров и фотографий прямо в чате.

## 💾 Схема базы данных (Database Schema)

Проект использует **SQLite** с полностью асинхронным драйвером `aiosqlite`. Логическая модель состоит из 4 связанных таблиц с поддержкой внешних ключей и каскадного удаления (`ON DELETE CASCADE`):


| Таблица | Первичный ключ | Назначение | Связи (Foreign Keys) |
| :--- | :--- | :--- | :--- |
| **`users`** | `tg_id` | Профиль пользователя и дата регистрации | — |
| **`notes`** | `note_id` | Текстовые заметки пользователя | `user_id` \(\rightarrow\) `users(tg_id)` |
| **`birthdays`** | `bday_id` | Имена именинников и даты праздников | `user_id` \(\rightarrow\) `users(tg_id)` |
| **`debts`** | `debt_id` | Финансовый баланс, тип долга и комментарий | `user_id` \(\rightarrow\) `users(tg_id)` |

## 🚀 Быстрый старт (Installation & Run)

### 1. Клонирование репозитория и подготовка
Создайте директорию на вашем ПК, откройте терминал и установите необходимые зависимости:
```bash
pip install -U aiogram pydantic-settings aiosqlite cachetools
```

### 2. Настройка конфигурации
Откройте файл `config.py` (или создайте файл `.env`) и вставьте туда токен вашего бота, полученный от [@BotFather](https://t.me):
```python
BOT_TOKEN = "ВВОДИТЬ СВОЙ ТОКЕН"
```

### 3. Запуск бота
Запустите главный файл архитектуры:
```bash
python main.py
```
